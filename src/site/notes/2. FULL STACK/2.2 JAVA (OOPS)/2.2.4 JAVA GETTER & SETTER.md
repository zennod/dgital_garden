#java
# GETTER 
- *The Getter is used to* **GET** *the value of an data member used in the constructor.*
```java
public int getRno(){
	return this.rno;
} 
```

# SETTER
- *The Setter is used to* **SET** *the value of an data member used in the constructor.*
```java
public void setRno(int r){
	rno = r
}
```
